RISC-V OS Kernel

Jan 2025 – May 2025
C GDB RISC-V UART Virtual Memory

Overview

Built a preemptive multitasking kernel entirely in C, incorporating a 100ms time slice round-robin scheduler to manage up to 16 concurrent processes.

Key Features

  • Implemented a complete virtual memory system using 3-level Sv39 paging, providing a 500MB virtual address space per process and managing memory in 2048B pages.
  • Developed an interactive shell that supports 4 commands and inter-process piping, enabling complex operations such as two simultaneous program executions.
